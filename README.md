# TypeScript Project starter

Simple started for TypeScript Node.js project.

## Running the project

1. Open terminal.
1. Type, "pnpm run dev".

## Type checking

1. Open terminal.
1. Type, "pnpm run check".

## Debugging

1. Set breakpoints.
1. Open debug panel.
1. Select "Debug Run".
1. Press F5 to run.

### Alternative debugging steps

1. Set breakpoints.
1. Open terminal.
1. Type, "pnpm run debug".
1. Press F5 to attach debugger.

## Updating dependencies

1. Open terminal.
1. Type, "pnpm run up".
